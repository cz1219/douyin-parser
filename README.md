## 🎉 首次发布：小晨万能解析 v2.0.0

### ✨ 主要功能
- 🎬 **多平台解析**: 支持抖音、快手、哔哩哔哩、小红书、微博、微视、西瓜视频、皮皮虾、QQ小世界、皮皮搞笑、最右、美拍、虎牙、梨视频、新片场、Twitch、Snapchat、Facebook、Vk、Instagram、YouTube、TikTok等20+主流平台
- 📥 **多画质下载**: 支持原画质、高清、标清等多种画质选择
- 🎵 **音乐提取**: 一键提取并下载视频背景音乐
- 📋 **智能链接**: 自动从复杂文本中识别各平台视频链接
- 📊 **进度显示**: 实时显示下载进度和状态
- 🎨 **精美UI**: Material Design 3设计，支持深色模式
- 🤖 **AI文案提取**: 使用先进AI模型提取视频中的文案内容
- 💾 **智能缓存**: 文案和视频信息自动保存，下次查看无需重新提取

### 🌟 支持平台列表
- 📱 **短视频**: 抖音、快手、微视、美拍、皮皮虾
- 📺 **长视频**: 哔哩哔哩、西瓜视频、虎牙直播、梨视频、新片场
- 📝 **社交**: 小红书、微博、QQ小世界、皮皮搞笑、最右
- 🌍 **国际**: TikTok、Instagram、Facebook、YouTube、Twitch、Snapchat、Vk

### 🤖 AI文案提取功能
- 支持4种先进识别模型：
  - **fun-asr-2025-08-25** (推荐模型，最新版本)
  - **paraformer-v2** (多语言支持)
  - **fun-asr** (高精度识别)
  - **sensevoice-v1** (情感识别)
- 文案自动缓存，提取后永久保存
- 支持在弹窗内切换模型重新提取
- 一键复制文案内容

### 🛠️ 技术特性
- 支持Android 8.0+ (API 26+)
- 全面适配Android 13+权限管理
- MVVM架构，代码清晰稳定
- 自动保存到相册/音乐文件夹
- 智能识别多种链接格式
- 批量解析多个视频链接
- Room数据库本地缓存

### 📱 使用方法
1. 复制任意支持平台的视频链接
2. 粘贴到应用输入框（支持多行输入）
3. 点击"解析视频"按钮
4. 选择下载视频、音乐或提取文案
5. 文案提取：点击"提取文案"→选择模型→等待完成→复制内容
