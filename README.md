# 抖音视频解析器

一款功能强大的抖音视频解析Android应用，支持视频下载、音乐下载、链接复制等功能。

## 功能特点

- 🎬 **视频解析**: 支持抖音视频链接解析，获取视频详细信息
- 📥 **视频下载**: 支持多种画质选择，可下载原画质视频
- 🎵 **音乐下载**: 支持提取并下载视频背景音乐
- 📋 **智能链接**: 自动从复杂文本中提取抖音链接
- 📊 **下载进度**: 实时显示下载进度和状态
- 🎨 **Material Design 3**: 现代化UI设计，支持深色模式
- 📱 **Android 13+**: 全面适配最新Android权限管理

## 下载安装

### 下载APK
请从Releases页面下载最新版本的APK文件。

### 安装步骤
1. 下载APK文件到手机
2. 在设置中允许"来自未知来源的应用"安装
3. 点击APK文件进行安装
4. 首次启动时授予必要的存储权限

## 使用说明

### 基本使用
1. 复制抖音视频链接
2. 粘贴到应用输入框
3. 点击"解析"按钮
4. 选择下载视频或音乐

### 智能链接识别
应用支持从复杂文本中自动识别抖音链接，例如：
```
@1.25 复制打开抖音，看看【小明的日常】今天天气真好，适合出去走走~ # 日常分享# 生活记录# 阳光正好 https://v.douyin.com/ABC123DEF456/ P@Y.nM LmN:/ 11/30
```

### 下载管理
- 实时显示下载进度
- 支持后台下载
- 自动保存到相册/音乐文件夹
- 下载失败自动重试

应用需要以下权限：

## 常见问题

### Q: 下载的视频在哪里？
A: 视频默认保存在手机相册的Movies文件夹中，音乐保存在Music文件夹中。

### Q: 为什么无法下载？
A: 请检查网络连接和存储权限，确保有足够的存储空间。

### Q: 支持哪些平台？
A: 目前主要支持抖音平台，后续可能扩展支持其他平台。

### Q: 应用安全吗？
A: 应用仅解析公开视频信息，不会获取用户隐私数据，所有下载文件仅在本地存储。
